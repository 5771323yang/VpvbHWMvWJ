# 医院信息管理系统 java655

## 项目概述

医院信息管理系统是一个综合性的管理平台，旨在提高医疗服务质量，优化医院运营流程。系统采用前后端分离的技术架构，后端提供稳定的数据处理能力，前端实现用户友好的交互界面。

## 技术栈

- 后端：Spring Boot，MyBatis
- 前端：Vue，ElementUI，Axios，ECharts，HTML，JavaScript，CSS
- 系统环境：IDEA，JDK 1.8，MySQL 8，Maven

## 功能模块

### 系统角色

系统分为以下三种角色：

- 管理员
- 医生
- 用户

### 前台功能

- 首页推荐信息：展示医院重要通知和推荐内容
- 医院概况：浏览医院的基本信息
- 医生信息展示：展示医生的专业信息
  - 医生信息浏览
  - 收藏医生
  - 评论互动
  - 在线挂号
- 个人中心
  - 查看和修改个人信息
  - 管理我的收藏

### 管理后台功能

- 个人中心
  - 修改密码
  - 查看和修改个人信息
- 用户管理
  - 查询、新增、修改、删除用户数据
- 医生管理
  - 查询、新增、修改、删除医生数据
- 科室管理
  - 查询、新增、修改、删除科室信息
- 医生信息管理
  - 查询、新增、修改、删除医生信息
- 挂号信息管理
  - 查询、新增、修改、删除挂号信息
  - 添加病例，支付操作
- 取消挂号管理
  - 查询、修改、删除取消挂号信息
- 问诊记录管理
  - 查询、新增、修改、删除问诊记录，支持打印
- 病房管理
  - 查询、新增、修改、删除病房信息
- 药房管理
  - 查询、新增、修改、删除药品信息，以及报表分析
- 系统管理
  - 查询、新增、修改、删除医院概况数据
  - 轮播图管理

## 运行环境

- 服务器：推荐使用支持 JDK 1.8 的服务器
- 数据库：MySQL 8
- IDE：IntelliJ IDEA

## 目录结构

```
hospital-management-system/
├── backend/              # 后端代码目录
├── frontend/             # 前端代码目录
└── db/                   # 数据库文件目录
```

## 部署步骤

1. 环境搭建：安装 JDK 1.8，MySQL 8，Maven，并配置相应环境变量。
2. 数据库准备：在 MySQL 中创建数据库，并导入 db 目录下的 SQL 文件。
3. 后端部署：编译后端代码，运行主程序。
4. 前端部署：编译前端代码，部署到 Web 服务器。

## 核心亮点

- 前后端分离，便于维护与扩展。
- 系统功能全面，覆盖医院管理及患者服务的多个方面。
- 稳定的后端处理能力，保障系统运行效率。
